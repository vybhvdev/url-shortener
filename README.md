🔗 Smart URL Shortener with Real-Time Analytics

A modern, production-ready URL shortener designed for performance, reliability, and actionable insights.
This platform not only creates short links instantly but also provides deep visibility into how, when, and where your links are used.

Built using Node.js, Express, and PostgreSQL (Supabase) with a scalable serverless deployment architecture.

---

🚀 Why This URL Shortener Matters

Most basic URL shorteners only reduce link length.
This system goes further by turning every click into useful data.

Key Benefits

📊 Actionable Analytics
